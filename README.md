# Airbnb-Listings-SQL-Analysis
Bu repository-də Airbnb listings dataseti üzərində MySQL istifadə edərək etdiyim analiz tapşırıqları yerləşir.
[SQL task  (Tərlan N.) 20260424.sql](https://github.com/user-attachments/files/27061892/SQL.task.T.rlan.N.20260424.sql)
📊 Analiz zamanı cavablandırılan əsas suallar:

Qiymət Analizi: Fərqli otaq növləri (room_type) və qonşuluq əraziləri üzrə orta, maksimum və minimum qiymət trendlərinin müəyyənləşdirilməsi.

Təklif Diversifikasiyası: Mülkiyyət növləri (property_type) və yerləşmə məkanlarına görə unikal elan saylarının statistikası.

Host Performansı: Ev sahiblərinin cavabvermə sürəti, qəbul dərəcələri və şəxsiyyəti təsdiq olunmuş hostların bazar payı.

Məkan Statistikası: Qonşuluq qrupları üzrə elanların sıxlığı və ən çox tələb olunan zonaların müəyyən edilməsi.

Filtrlənmiş Axtarışlar: "Superhost" statusu, minimum gecələmə sayı və spesifik ad nümunələrinə (Pattern Matching) görə dərinləşdirilmiş filtrasiya.

🛠 İstifadə olunan Texniki Bacarıqlar:
Aqreqasiya Funksiyaları: SUM(), AVG(), COUNT(), MIN(), MAX().

Məlumatın Qruplaşdırılması: GROUP BY, HAVING və nəticələrin limitlənməsi (LIMIT).

Mətn funksiyaları: CONCAT(), SUBSTR(), LIKE operatoru ilə mürəkkəb axtarışlar.

Sıralama və Yuvarlaqlaşdırma: ORDER BY və ROUND() funksiyaları.
